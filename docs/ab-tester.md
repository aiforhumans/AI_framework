# A/B Tester Tab

## Overview
Side-by-side comparison tool for testing multiple models or agents simultaneously.

## Features

### Variant Management
- Add multiple models
- Add multiple agents
- Mix models and agents
- Remove variants

### Testing Configuration
- Single prompt input
- System prompt support
- Same settings for all variants
- Parallel execution

### Results Display
- Side-by-side response cards
- Latency tracking per variant
- Error handling per variant
- Visual comparison layout
